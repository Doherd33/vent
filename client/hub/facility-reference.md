# Facility Reference — WuXi Biologics Ireland (Dundalk)

## Site Overview

**Location:** Dundalk Science and Technology Park, Mullagharlin, Co. Louth, A91 X56F
**Buildings:**
- DK1 — Admin Building (Reception, Training, Offices, QC Labs)
- DK2 — Manufacturing (MFG 6.1, MFG 7, MFG 6.2)
- DK3 — Warehouse (GMP Released)
- DK4 — CUB (Central Utilities Building)
- DK5 — Utility
